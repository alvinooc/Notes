2. 注意：

   - 默认高度是40px

   - 自定义高度要修改内部css

   - **右侧的箭头也要修改，否则会有偏移问题**

     ![image-20220415203409371](https://cdn.jsdelivr.net/gh/alvinooc/Notes@master/Images/image-20220415203409371.png)

   * el-select 展开列表的 z-index 默认为 2001，所以要注意它父级的 z-index，否则会在下面

     ![image-20220415203425188](https://cdn.jsdelivr.net/gh/alvinooc/Notes@master/Images/image-20220415203425188.png)

3. 解决方式

   ```js
   /* 自定义宽度 */
   .my-el-select {
   	width: 320px;
   
   	/* select 框的高度设置，默认是 40px */
   	.el-input__inner {
   		height: 30px;
   	}
   
   	/* 设置右侧按钮居中 */
   	.el-input__icon {
   		line-height: inherit;
   	}
   	
   	.el-input__suffix-inner {
   		display: inline-block;
   	}
   }
   ```
